Frage

Was ist eine Menge?

Antwort

Eine Menge ist ein Verbund, eine Zusammenfassung von einzelnen Elementen. Diese können unter einem Oberbegriff zusammengefasst werden:

  • Menge der Kontinente
  • Menge der ganzen Zahlen
  • Menge der Mädchennamen
Nachlesen

Frage

Wie können Mengen dargestellt werden?

Antwort

Mengen können auf verschiedene Art dargestellt werden:

  • Liste (z.B.: Kontinente: Europa, Afrika, Asien, etc.)
  • Mengendiagramm (Graphische Darstellung als Blase)

Frage

Was ist eine Zuordnung?

Antwort

Bei einer Zuordnung wird jedes Element der Ausgangsmenge mit einem oder mehreren Elementen der Zielmenge verknüpft.

  • Land mit Kontinent
  • Person mit Schuhgröße
  • Zahl mit seinen Teilern

Frage

Wie können Zuordnungen dargestellt werden?

Antwort

  • Land mit Kontinent
